## Deployment

To deploy this project

Set in .env your domain name,

example:

```bash
  DOMAIN_NAME=`localhost`
```

```bash
  docker-compose up -d
```


## Run Locally

Clone the project

```bash
  git clone https://github.com/Globtermic/Web-app
```

Go to the project directory

```bash
  cd Web-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
